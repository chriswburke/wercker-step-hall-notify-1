# hall-notify

POST request to Hall(.com)

## Example:

```
  after-steps:
      - tgfjt/hall-notify@0.1.7:
          token: HALL_ROOM_API_TOKEN
          icon: YOUR_ICON_URL
```

## Options:

* `token`: HALL_ROOM_API_TOKEN **required**
* `icon`: YOUR_ICON_URL(on Hall Chat)
